# Freelancer Toolkit - Backend

The **Freelancer Toolkit Backend** is a Node.js application that powers the backend of the Freelancer Toolkit web app. It handles API integrations, data processing, and communication with external services such as currency exchange, places, and time zone APIs.

### APIs Integrated:
- **Currency Exchange API**: Convert expenses into your preferred currency for organized record-keeping.
  - [Currency Exchange API Documentation](https://www.exchangerate-api.com/docs/overview)
- **Places API**: Find local freelancer-friendly spots like cafes and coworking spaces with Wi-Fi, ratings, and amenities.
  - [Places API Documentation](https://developers.google.com/maps/documentation/places/web-service/overview)
- **Time Zone API**: Manage time zones for accurate meeting scheduling with international clients.
  - [Time Zone API Documentation](https://developers.google.com/maps/documentation/timezone/overview)

### Features:
- **Currency Conversion Endpoint**: API endpoint to convert amounts between different currencies.
- **Workspace Finder Endpoint**: API to search for nearby freelancer-friendly workspaces.
- **Time Zone Handling Endpoint**: API for converting times between different time zones.
